# Wisecar – אתר לוח רכבים (עברית + ₪)

האתר בנוי מחדש בסגנון דומה לדוגמאות ששלחת:
- דף בית עם Hero + חיפוש, קטגוריות, ורכבים אחרונים
- עמוד רכבים עם חיפוש מתקדם (פילטרים) וקארדים גדולים
- עמוד פרטי רכב עם גלריה ועמודת מחיר/יצירת קשר

## הפעלה מקומית

1. התקנה:
```bash
npm install
```

2. הרצה:
```bash
npm run dev
```

## הגדרות Supabase
יש להגדיר משתני סביבה:
- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`

אפשר לשים ב-`.env` או ב-`public/env.js`.

## טלפון/וואטסאפ (אופציונלי)
אפשר להגדיר:
- `VITE_CONTACT_PHONE`
- `VITE_WHATSAPP_PHONE`

אם לא מוגדר – ברירת מחדל: `050-000-0000`.
